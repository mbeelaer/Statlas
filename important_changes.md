# Important changes

This document is a log of important changes in how the Statlas website is created and managed. Dates in YYYY-MM-DD format.

## 2025-11-06

From now on, demo pages will be made using Quarto instead of RMarkdown. The source files needed to render these pages will consist of

- a .qmd file with a highly specific YAML configuration
- a .html file containing CSS styling rules
